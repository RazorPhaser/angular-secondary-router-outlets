A demonstration of Secondary (Named) router outlets and some gotcha's.

Secondary Router Outlets:

  • Must be resolved at the module where they are being used

  • Must NOT be part of a route configuration that uses a componentless element

  • Declaring paths may help to resolve the route to the component, BUT the named router-outlet will only be populated at the same level in the heirachy in which it is resolved.
